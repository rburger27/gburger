Hands-On: Robot Store Day Three
Now that you have an understanding, and have seen some Bootstrap in action, it's time to put that knowledge to work. In this Hands-On exercise you will be adding Bootstrap to the Robot Store created in the previous Hands-On projects. Hands-On projects are completed outside of the browser, using your IDE, or editor of choice.

Description:
You should leverage what you have learned about Bootstrap as follows:

You should have the Robot Store on your computer from the previous Hands-On projects. From this, you will need to add some functionality using jQuery to meet the requirements below.

Requirements
Convert your Navbar to a Bootstrap Navabar
Add a collapsible Navbar for using in Mobile and Tablet devices
Tip! You can see an example of this here. Resize the page to see how the Navbar interacts with the sizing.
Make your site Responsive by implementing the Bootstrap Grid System.
Make your Robots into Thumbnails with captions that contain the information given to you in the HTML.
